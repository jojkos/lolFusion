# M6 — Emoji-Grid Share Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Replace the plain-text share with a spoiler-free, Wordle-style emoji block that conveys how the puzzle went (misses, the two finds, bonus), plus streak, score, and a play URL — with no champion or skin-line names.

**Architecture:** A pure, unit-tested `lib/share.ts#buildShareText(...)` produces the share string. `GameInterface`'s `onShare` calls it with the live values + `window.location.origin`. Builds on M1–M5 (branch `feat/skinline-bonus`).

**Tech Stack:** Next.js 16, React 19, TypeScript, vitest.

## Global Constraints

- The share text must be **spoiler-free**: no champion names, no skin-line/theme name.
- Win grid: one `🟦` per champion-phase miss (`max(0, attempts - 2)`) followed by `🟩🟩` for the two champions found; append ` ✦` when the bonus was solved. Surrender: `🟥 Surrendered`.
- Include `🔥 {streak} day streak` only when `streak > 0`; include `Score {score}` only when not surrendered; always end with the play URL.
- `npx tsc --noEmit` clean; `npm test` green. Do NOT run `npm run build`.

- [ ] **Step 3: Implement `lib/share.ts`**

```ts
export function buildShareText(o: {
  date: string;
  attempts: number;
  givenUp: boolean;
  bonusSolved: boolean;
  streak: number;
  score: number | null;
  url: string;
}): string {
  const lines: string[] = [`LoL Fusion · ${o.date}`];

  if (o.givenUp) {
    lines.push('🟥 Surrendered');
  } else {
    const misses = Math.max(0, o.attempts - 2);
    lines.push('🟦'.repeat(misses) + '🟩🟩' + (o.bonusSolved ? ' ✦' : ''));
  }

  if (o.streak > 0) lines.push(`🔥 ${o.streak} day streak`);
  if (!o.givenUp && o.score !== null) lines.push(`Score ${o.score}`);

  lines.push(o.url);
  return lines.join('\n');
}
```
